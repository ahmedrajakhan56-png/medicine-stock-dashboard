## 🌟 **Project Overview**

A comprehensive **medicine stock management system** designed to revolutionize pharmacy inventory tracking. This dashboard provides **real-time visibility** into stock levels, automated alerts for low inventory, and intelligent expiry monitoring to ensure patient safety and operational efficiency.

---

## ✨ **Key Features**

### 🚨 **Smart Alert System**
- **Low Stock Detection** - Automatic alerts when medicines fall below reorder levels
- **Expiry Risk Monitoring** - 30/60/90 day expiry warnings with color-coded risk levels
- **Emergency Stock Tracking** - Special monitoring for critical life-saving medicines

### 📊 **Advanced Analytics**
- **Category-wise Distribution** - Visual breakdown of tablets, injections, syrups, and more
- **Supplier Performance** - Track reliability, delivery times, and order history
- **Sales Trends** - Monthly, quarterly, and yearly sales patterns

### 🎯 **Interactive Dashboard**
- **Real-time Filters** - Filter by category, supplier, expiry date, or stock status
- **Drill-down Capabilities** - Click from summary to detailed medicine-level data
- **Mobile Responsive** - Access insights on any device

## 🚀 **Getting Started**

### Prerequisites
```bash
# Install required Python packages
pip install pandas numpy matplotlib seaborn plotly
```

### Database Setup
```sql
-- Execute SQL scripts to create database schema
CREATE DATABASE MedicineStockDB;
USE MedicineStockDB;
-- Run provided schema.sql file
```

### Running the Dashboard
1. Clone the repository
2. Generate sample data using Python scripts
3. Import data to SQL Server
4. Open Power BI dashboard file
5. Refresh data connections
6. Explore insights!
